## 1. 现有设计分析
当前黑色主题存在的问题：
- **品牌一致性不足**：纯黑背景可能与实验室管理的专业、洁净形象不符，且容易显得压抑。
- **可读性问题**：低对比度的文字在黑色背景下可能难以辨识，尤其是长时间使用时易造成视觉疲劳。
- **无障碍缺陷**：可能未达到WCAG 2.1 AA标准，对视力障碍用户不友好。
- **层级感缺失**：缺乏明显的分隔和状态反馈，导致导航结构不清晰。

## 2. 新配色方案设计
制定一套符合现代SaaS风格的配色方案，兼顾专业性与舒适度。

### 2.1 核心色彩 (Color Palette)
- **主色 (Primary)**: `#3B82F6` (Blue-500) - 代表科技、信任与专业。
- **辅色 (Secondary)**: `#10B981` (Emerald-500) - 用于成功状态或强调。
- **背景色 (Background)**:
  - **亮色模式**: `#FFFFFF` (White) / `#F8FAFC` (Slate-50) - 侧边栏背景。
  - **暗色模式**: `#1E293B` (Slate-800) - 深蓝灰色，比纯黑更柔和。
- **文字色 (Text)**:
  - **主标题**: `#1E293B` (Slate-800) / `#F1F5F9` (Slate-100)
  - **次级文本**: `#64748B` (Slate-500) / `#94A3B8` (Slate-400)
- **交互色 (Interaction)**:
  - **悬停 (Hover)**: `rgba(59, 130, 246, 0.1)` (亮) / `rgba(59, 130, 246, 0.2)` (暗)
  - **选中 (Active)**: `#EFF6FF` (Blue-50) + 主色文字 (亮) / `#1E293B` + 主色左边框 (暗)

### 2.2 视觉风格
- **Glassmorphism (轻量应用)**：在折叠菜单或浮层中使用轻微的背景模糊 (`backdrop-filter: blur(8px)`)，增加现代感。
- **阴影与层级**：使用柔和的阴影 (`box-shadow: 4px 0 24px rgba(0,0,0,0.02)`) 代替硬边框，营造悬浮感。

## 3. 设计实现方案

### 3.1 CSS 变量定义 (CSS Variables)
我将创建一个 `sidebar-theme.css` 文件，定义亮暗两套主题变量。

```css
:root {
  /* Light Theme */
  --sidebar-bg: #FFFFFF;
  --sidebar-text: #334155;
  --sidebar-text-active: #2563EB;
  --sidebar-bg-hover: #F1F5F9;
  --sidebar-bg-active: #EFF6FF;
  --sidebar-border: #E2E8F0;
  --sidebar-width: 260px;
  --sidebar-collapsed-width: 64px;
  --transition-speed: 0.3s;
}

[data-theme='dark'] {
  /* Dark Theme */
  --sidebar-bg: #0F172A;
  --sidebar-text: #94A3B8;
  --sidebar-text-active: #60A5FA;
  --sidebar-bg-hover: #1E293B;
  --sidebar-bg-active: #1E293B;
  --sidebar-border: #1E293B;
}
```

### 3.2 组件结构与交互
- **响应式设计**：
  - **桌面端 (>1024px)**: 默认展开，支持折叠。
  - **平板端 (768px - 1024px)**: 默认折叠 (Icon-only)，悬停展开。
  - **移动端 (<768px)**: 隐藏，通过汉堡菜单唤出抽屉式导航。
- **动效参数**：
  - **折叠/展开**: `transition: width 0.3s cubic-bezier(0.4, 0, 0.2, 1)`
  - **图标切换**: `transform: scale(1.1)`，时长 `0.2s`

### 3.3 可访问性 (Accessibility)
- **对比度**：确保所有文本与背景对比度至少为 4.5:1 (AA级)。
- **键盘导航**：支持 `Tab` 键遍历菜单项，`Enter` 键激活。
- **ARIA 属性**：为导航区域添加 `role="navigation"`，为折叠按钮添加 `aria-expanded` 和 `aria-label`。

## 启动实验室管理系统计划

### 项目概述

这是一个基于Django的实验室管理系统，包含以下主要功能模块：

* 用户与角色管理

* 试验任务管理

* 试验数据管理

* 设备管理

* 工装管理

* 工具管理

* 报告生成

### 系统依赖

* **后端框架**: Django 4.2+

* **数据库**: MySQL (默认) 或 SQLite

* **缓存**: Redis

* **异步任务**: Celery + Redis

* **Web服务器**: Gunicorn (生产环境)

#### 3. 服务启动流程

1. **启动Redis服务** (用于缓存和Celery消息队列)
2. **数据库迁移** (如有需要)
3. **启动Celery Worker** (处理异步任务)
4. **启动Celery Beat** (定时任务调度)
5. **启动Django开发服务器**

#### 4. 启动命令

```bash
# 1. 激活虚拟环境
.venv\Scripts\activate

# 2. 启动Redis (需要单独安装Redis服务)
# redis-server

# 3. 数据库迁移 (首次启动或模型变更时)
python manage.py migrate

# 4. 启动Celery Worker
celery -A laboratory_management worker -l info

# 5. 启动Celery Beat (可选，用于定时任务)
celery -A laboratory_management beat -l info

# 6. 启动Django开发服务器
python manage.py runserver 0.0.0.0:8000
```
